Statut API

Cette application diffuse du contenu stocké dans Azure Blob Storage via une API REST déployée sur Kubernetes.


Endpoints disponibles :
  • /api/events
  • /api/news
  • /api/faq
  • /healthz
  • /readyz

Les données sont mises en cache côté serveur pour réduire les accès au stockage et améliorer les performances.
